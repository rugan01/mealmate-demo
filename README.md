# MealMate Demo

A colourful, clickable school-meal booking prototype created for a Grade 6 Mock Shark Tank presentation.

## The idea

MealMate replaces paper breakfast and lunch coupons with a simple digital flow:

1. Parents view the weekly menu.
2. They choose breakfast, lunch, or both.
3. The school automatically receives the booking.

The prototype uses fictional demonstration data and does not process real payments or student information.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

GitHub Pages deployment is handled automatically through GitHub Actions.
